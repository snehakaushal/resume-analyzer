Python, Flask, NLP, Machine Learning, HTML, CSS, JavaScript

Developed an AI-powered Resume Analyzer web application that evaluates resumes for ATS compatibility and provides intelligent improvement suggestions.
Built resume parsing functionality using Python and PyPDF2 to extract resume content from uploaded PDF files.
Implemented ATS score calculation based on resume sections, keyword matching, formatting quality, content depth, and technical skills analysis.
Added AI-based Job Description matching system using TF-IDF and cosine similarity techniques to calculate resume-job compatibility scores.
Designed a Skill Gap Analysis module to identify missing keywords and required skills from job descriptions.
Developed grammar and resume quality analysis features to detect writing mistakes, weak phrasing, and formatting issues.
Integrated automatic section detection for Education, Skills, Experience, Projects, Certifications, Achievements, and Summary sections.
Created an AI Interview Preparation module that generates technical and HR interview questions based on extracted resume skills.
Built a Resume Builder feature that generates tailored resumes according to specific job descriptions and required technologies.
Used Flask for backend development and integrated libraries including PyPDF2, scikit-learn, and Werkzeug for document processing and analysis
